Git commands

  • git init : To initialize git
  • git add filename: to add the particular file
  • git add. : to add everthing
  • git commit -m 'some message': to keep track we use commit
  • git status: to check status
  • git log: to get back to previous version we can see all the logs here
  • git reset --hard "paste the thing copied from log here" =>will delete the code
  • git reset --soft "paste the thing copied from log here" =>will not delete the code
